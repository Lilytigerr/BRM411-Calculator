# 快速開始指南

## 🚀 最快上傳到 GitHub 的方法（5 分鐘完成）

### 第一步：準備檔案
你需要的檔案（全部都在下載區）：
- ✅ `BRM411_Calculator_Final.html` - 主程式
- ✅ `README.md` - 說明文件
- ✅ `.gitignore` - Git 忽略檔案（可選）

### 第二步：登入 GitHub
1. 前往 https://github.com
2. 如果沒有帳號，點「Sign up」註冊
3. 如果有帳號，點「Sign in」登入

### 第三步：創建新專案
1. 點擊右上角「+」號
2. 選擇「New repository」
3. 填寫：
   ```
   Repository name: BRM411-Calculator
   Description: BRM411 臨床試驗訪視計算器
   ```
4. 選擇「Public」（公開）
5. **不要**勾選「Add a README file」（我們已經有了）
6. 點擊「Create repository」

### 第四步：上傳檔案
1. 在新頁面，點擊「uploading an existing file」
2. 拖曳以下檔案到上傳區：
   - `BRM411_Calculator_Final.html`
   - `README.md`
   - `.gitignore`
3. 在「Commit changes」填寫：
   ```
   Initial upload - BRM411 Calculator
   ```
4. 點擊「Commit changes」

### 第五步：啟用網頁（讓別人可以直接用）
1. 點擊上方的「Settings」
2. 左側找到「Pages」
3. 在「Branch」選擇：
   - Branch: `main`
   - Folder: `/ (root)`
4. 點擊「Save」
5. 等待 1-2 分鐘

### 第六步：取得網址
1. 重新整理頁面
2. 會看到綠色訊息：
   ```
   Your site is live at https://[你的名字].github.io/BRM411-Calculator/
   ```
3. 點擊網址測試

### 完成！🎉

你的網址是：
```
https://[你的GitHub用戶名].github.io/BRM411-Calculator/BRM411_Calculator_Final.html
```

## 💡 讓網址更簡潔（可選）

## 🔄 如何更新

### 方法 1：直接編輯（最簡單）
1. 在 GitHub 找到要修改的檔案
2. 點擊檔案名稱
3. 點擊右上角鉛筆圖示（Edit）
4. 修改內容
5. 點擊「Commit changes」

### 方法 2：上傳新版本
1. 在 Repository 頁面點擊「Add file」→「Upload files」
2. 上傳新版本的檔案（會自動覆蓋舊的）
3. Commit changes

## ❓ 常見問題

### Q: 網頁顯示 404
**A**: 等待 2-5 分鐘讓 GitHub Pages 建置完成

### Q: 更新後沒變化
**A**: 清除瀏覽器快取（Ctrl+F5）

### Q: 想刪除專案
**A**: Settings → 最下方 Danger Zone → Delete this repository

### Q: 想改成私人的
**A**: Settings → Danger Zone → Change visibility → Make private
（注意：私人專案的 GitHub Pages 需要付費）
