# BRM411 訪視計算器

BRM411 臨床試驗訪視日期規劃工具

## 功能特色

✅ **智能日期驗證**
- 自動驗證 Naive 和 Washout 受試者的日期間隔
- 即時檢查週末和台灣國定假日 (2025-2027)
- 動態顯示驗證結果和建議

✅ **藥物洗除期管理**
- Prostaglandins / β-blockers / Kinase inhibitors (28 天)
- Adrenergic agonists (14 天)
- Muscarinic agonists / Carbonic anhydrase inhibitors (7 天)

✅ **自動計算功能**
- Visit 2 建議（最短符合時間）
- Visit 1a 範圍計算和建議
- 完整訪視排程生成 (V1, V1a, V2, Day 2, V3-V6)

✅ **即時假期提醒**
- 輸入日期後立即顯示週末/假期警告
- 支援 2025-2027 台灣國定假日
- 週末和假日雙重檢測

## 使用說明

### Naive 受試者
1. 選擇「Naive (未用藥受試者)」
2. 輸入 Visit 1 日期（篩選日）
3. 輸入 Visit 2 日期（基線日）
4. V1 到 V2 需要 1-35 天
5. V1a 可在 V1 和 V2 之間任選（通常與 V1 同天）

### Washout 受試者
1. 選擇「Washout (洗除期受試者)」
2. 選擇需洗除的藥物類型
3. 輸入 Visit 1 日期（篩選日）
4. 輸入 Visit 2 日期（基線日）
5. V1 到 V1a 必須等於洗除期天數
6. V1a 到 V2 必須是 1-7 天

## 台灣國定假日（2025-2027）

### 固定假日
- 元旦：1/1
- 和平紀念日：2/28
- 兒童節：4/4
- 清明節：4/5（依節氣調整）
- 勞動節：5/1
- 國慶日：10/10
- 行憲紀念日：12/25

### 農曆假日
- 春節：除夕到初六（共 7 天）
- 端午節：農曆五月初五
- 中秋節：農曆八月十五
